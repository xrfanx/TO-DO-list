یک برنامه ریز حرفه ای و زیبا برای یاد اوردی کارها 


# لیست کارها (To-Do List) ساده و زیبا
یک وب پیج مینیمال و ساده که با استفاده از html css js نوشته شده است

این پروژه امکان اپدیت شدن را دارد

## ویژگی‌ها

- اضافه کردن کار جدید با دکمه یا Enter  
- حذف کار با دکمه سطل زباله  
- نمایش هشدار اگر فیلد خالی باشه  
- طراحی مینیمال و زیبا با رنگ آبی جذاب  
- اسکرول خودکار وقتی تعداد تسک‌ها زیاد بشه  
- کپی‌رایت و لینک گیت‌هاب در پایین صفحه  
- کاملاً responsive (مخصوصاً برای عرض زیر ۴۲۰px بهینه شده)


## تکنولوژی‌ها

- HTML5  
- CSS3 (Flexbox + Media Queries)  
- Vanilla JavaScript (بدون کتابخونه خارجی)
